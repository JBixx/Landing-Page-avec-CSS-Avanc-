# PulseNet — Landing Page

Projet 3.3 de l'Akieni Academy : landing page avec CSS avancé.

PulseNet est une page de présentation pour un outil de surveillance réseau. Le site est entièrement fait en HTML et CSS, sans JavaScript.

## Sections

- Hero
- Fonctionnalités (3 colonnes)
- Avis clients
- Tarifs (en FCFA)
- Appel à l'action (CTA)
- Footer

## Techniques CSS utilisées

- Variables CSS (`--primary-color`, etc.)
- Sélecteurs combinateurs (`+`, `~`, `>`)
- Pseudo-classes (`:nth-child`, `:first-of-type`)
- Reset CSS personnalisé
- Flexbox et Grid
- Design responsive

## Structure

```
landing/
└── folder/
    ├── index.html
    └── style.css
```

## Comment voir le projet

Ouvrir le fichier `landing/folder/index.html` dans un navigateur.
